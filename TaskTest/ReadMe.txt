Программа для тестирования задач на валидацию с выводом.
 С условием наличия правильного решения
В класс Solution пишется код решения задачи
В класс TrueSolution загружается правильное решение. 
В первом режиме при запуска класса test пользователь сам вводит данные.
Во втором режиме класс test генерирует случайным образом задданное количество чисел.Величина чисел также задается пользователем
После запуска, сравнивает два вывода и если вывод Solution отличается от 
TrueSolution, то программа завершается и выводит в консоль выводы классов, 
а также поток ввода при котором возникает ошибка.
